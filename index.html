Temporal Difference General Game Player

Rather than writing an program that can play a single game well, why not write an AI that can play any game well?

My program is able to read games in XML format (The XML must follow the included game.xsd grammar). You can run the game by running start.py It will then allow you to select the players (human, random, or AI), at which point, it will run the game. The AI needs training to learn how to play well. To train the AI, run learningplayer.py.

A sample game, Dominion, has been included, and is described in dominion.py.
